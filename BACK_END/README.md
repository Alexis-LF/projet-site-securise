# Back end (api)
## liste des endpoints :
### Récupérer la liste des villes
type de requête : *GET*

```
/index.php/villes
```
### Récupérer la liste des professions
type de requête : *GET*

```
/index.php/professions
```
### Recherche par nom et professions et ville
type de requête : *GET*

```
/index.php/recherche?nom=NOM&profession=PROFESSION&ville=VILLE
```
